# 更新公告素材(2026-07-11 ~ 07-15,近 30 次提交提炼)

> 只留用户可感知的新特性与重要修复,写公告时按需取用。
> 内部工程(混淆强化/服务器部署/文档调研/脚本)已略去。

## 重磅新功能

- **无线遥控 · 实时监看**:手机取景、录像模式监看、HD 高清取景、帧率显示
  - 数值**上下拖拽调参**(快门/光圈/ISO/曝光补偿,一格一档带触感,点击弹全表直跳)
  - **两段式快门**:按住对焦(合焦震动 + 对焦框转绿)、松开拍摄、滑出取消
- **大文件支持**:分块传输 + 断点续传,4GB 以上视频也能传
- **拍一张,列表立刻出现**:连接期间按快门(机身或遥控拍摄),新照片实时插入照片列表
- **连拍自动识别**:连拍序列带青绿角标,一键筛选,不再被几百张相似照片淹没
- **保护标记(🔑)**:机内保护过的照片带钥匙角标并可筛选——相机里选片,手机只看选中的
- **筛选系统**:类型 / 保护 / 连拍,可叠加,跨会话记忆
- **长按预览大升级**:FHD 高清大图(模糊渐进加载)、双指缩放 / 双击放大、
  显示拍摄参数(光圈/快门/ISO/焦距)与**对焦点位置**;预览中同步展示连拍/保护标记

## 体验与视觉

- **入队动画**:整组传输时照片们"打包成摞"飞进队列按钮、单张化作小相纸飞入,
  队列按钮接住弹跳、落袋才跳数
- 队列按钮常驻,随时一键进队列页
- 传输卡片重排:状态一目了然,重试一键直达;完成卡片显示 **大小 · 速度 · 耗时**
- 缩略图状态角标重做:传输中显示**真实进度环**
- 全局按压手感、玻璃面板质感、三页视觉统一;连接速度更快(成功停留缩短一半)

## 重要修复

- 修复多处可能导致**文件损坏**的下载/续传缺陷:续传错位写入、短读留洞、
  超 4GB 视频截断却被标记完成、同名不同文件被拼接——现在完整性校验兜底
- 修复重连/换卡后预览的拍摄参数与对焦点**串到别的照片**
- 修复部分机型预览不显示光圈值
- 相机断联时的报错从系统原文改为人话提示(中/繁/英),提示可续传
