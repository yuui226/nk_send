## 三、操作步骤

1. 官网注册个人账号并实名认证;
2. 在线填报 R11 申请表(照抄上表)→ 系统生成 PDF 申请表 → 打印,申请人签名;
3. 打印源程序材料(60 页)、操作说明书(补好截图);
4. 连同身份证复印件(签名)一起邮寄至中国版权保护中心(官网有地址),或在部分城市的登记大厅现场提交;
5. 等待受理通知与证书(纸质证书邮寄到付)。

## 四、注意事项

- 三份打印材料均不要装订,用回形针/夹子即可;
- 源程序与说明书的页眉(软件名+版本号+页码)已在生成的 HTML 中自带,打印时**关闭浏览器自身的页眉页脚**(打印对话框-更多设置-取消勾选"页眉和页脚");
- 软件全称一旦提交不可随意更改,商标顾虑的话可先确认"Z传"无冲突;
- 申请期间不影响 App 开发迭代,证书对应的是申报版本。
