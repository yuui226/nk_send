服务端只做三件事

1. 返回当前版本

GET /v1/app/latest

2. 解析当前版本直链

POST /v1/app/download-url

3. 管理与统计

发布、软/硬切换、聚合计数。

只有当前版本

没有额外状态。

更新统计

不保存用户或设备信息。

检查成功

对应版本的检查次数 +1。

打开系统安装器

对应版本的安装触发次数 +1。

必要判断

检查更新

配置有效 → 返回；无效 → NO_VERSION_INFO。

请求下载

versionCode 必须等于当前发布版本，防止接口变成公共解析器。

解析结果

只接受 success=true 且 data.directLink 为 HTTPS。

自动更新失败：复制蓝奏云链接和密码，用户打开下载页。
